
About
=====

This is extensiont to `docutils` to provide some useful new directives.


New directives
==============

Blit.tv
-------

Pass the id of blip tv movie from embeded code, NOT from the link of the video to bliptv directive.
Other options:
 * 

Example::

    .. bliptv:
